# House Prices Data Preprocessing
This project performs complete preprocessing and exploratory data analysis on 
the House Prices dataset.
# Steps Included
- Data Loading
- Missing Value Handling
- Outlier Removal
- Exploratory Data Analysis
- Feature Engineering
- Data Encoding
- Pipeline Construction
# How to Run
1. Download train.csv and test.csv
2. Place them inside a data/ folder
3. Run the notebook from start to finish
